# 視覺設計原則

> 供 ux-design skill 的 Step 2（Research Design Patterns）參考。

## C.A.R.P. 四大設計原則

| 原則 | 說明 | 實踐 |
|------|------|------|
| **Contrast** 對比 | 不同重要性的元素要有明顯視覺差異 | 標題/內文的字級、粗細、色彩差異 |
| **Alignment** 對齊 | 每個元素都要與其他元素有視覺連結 | 統一左對齊 / 格線對齊，避免隨意置中 |
| **Repetition** 重複 | 重複使用視覺元素建立一致性 | 統一的按鈕樣式、卡片格式、間距規則 |
| **Proximity** 親密性 | 相關的資訊要在視覺上靠近 | 表單標籤緊鄰輸入框，無關元素拉開距離 |

## 色彩理論

### 60-30-10 配色法
- **60% 主色**：背景、大面積底色（通常是中性色：白/灰/深色）
- **30% 輔色**：區塊背景、卡片、次要元素
- **10% 強調色**：CTA 按鈕、重要標記、品牌色

### 色彩搭配模式
| 模式 | 說明 | 適用 |
|------|------|------|
| 單色系 | 同一色相不同明度/彩度 | 專業、穩重的 B2B 產品 |
| 互補色 | 色相環對面的兩色（例如藍+橙） | 需要強烈對比的 CTA |
| 類似色 | 色相環相鄰的 2-3 色 | 和諧、舒適的整體感 |
| 三角色 | 色相環上等距三色 | 活潑、多彩的 C 端產品 |

### 語義色彩
| 用途 | 色彩 | 說明 |
|------|------|------|
| 成功 | 綠色 | 完成、通過、上線 |
| 警告 | 黃色/橙色 | 注意、待處理 |
| 錯誤/危險 | 紅色 | 錯誤、刪除、逾期 |
| 資訊 | 藍色 | 提示、連結、選中 |
| 中性 | 灰色 | 禁用、次要、輔助 |

## 排版系統

### 字級階層（Type Scale）
建議使用 1.25 倍率（Major Third）或 1.333 倍率（Perfect Fourth）：

| 層級 | 用途 | 範例（base 16px） |
|------|------|---|
| Display | 頁面主標題 | 32px Bold |
| H1 | 區塊標題 | 24px Semibold |
| H2 | 子區塊標題 | 20px Semibold |
| Body L | 主要內文 | 18px Regular |
| Body | 一般內文 | 16px Regular |
| Caption | 輔助說明、時間戳 | 14px Regular |
| Micro | 標籤、badge | 12px Medium |

### 行高
- 標題：1.2 ~ 1.3
- 內文：1.5 ~ 1.6
- 行動裝置可適度收緊到 1.4

## 間距系統

### 4pt Base Grid
所有間距值為 4 的倍數：

| Token | 值 | 用途 |
|-------|------|------|
| space-xs | 4px | 密集排列（badge 內距） |
| space-sm | 8px | 相關元素間距（icon 與文字） |
| space-md | 12px | 元件內距（按鈕 padding） |
| space-lg | 16px | 區塊內元素間距 |
| space-xl | 24px | 區塊間距 |
| space-2xl | 32px | 大區塊間距 |
| space-3xl | 48px | 頁面級間距 |

### 留白策略
- **內容密集型**（表格、儀表板）：使用較小間距（8-12px），讓更多資訊可見
- **展示型**（著陸頁、介紹頁）：使用較大間距（24-48px），營造呼吸感
- **原則**：同一頁面內間距層級不超過 4 種，保持節奏一致

## 視覺掃描模式

### F-Pattern（列表頁 / 內容頁）
```
████████████████████
████████████████
████████████
████████
```
- 使用者先水平掃描頂部 → 往下掃描左側 → 重要操作放左上方

### Z-Pattern（著陸頁 / 行銷頁）
```
████████████████████
              ↗
████████████████████
```
- Logo 左上 → 導航右上 → 視覺焦點左下 → CTA 右下

## Accessibility 基準

| 規則 | 標準 | 驗證 |
|------|------|------|
| 文字對比度 | ≥4.5:1（一般文字）、≥3:1（大文字） | WebAIM Contrast Checker |
| 觸控目標 | ≥44×44pt（iOS）、≥48×48dp（Android） | 直接量測 |
| 可縮放文字 | 不使用固定 px，支援系統字級調整 | 測試 Dynamic Type |
| 色彩不是唯一提示 | 錯誤不只靠紅色，加 icon 或文字 | 灰階模式檢查 |
| 螢幕閱讀器 | 所有互動元素有 label | VoiceOver 測試 |
