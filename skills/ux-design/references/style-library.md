# 視覺風格庫

> 供 ux-design skill 的 Step 2 參考。配合 `ux-config.yaml` 的 `brand.style` 欄位使用。

## 風格選擇指引

| 產品類型 | 推薦風格 |
|---------|---------|
| B2B SaaS / 管理後台 | Minimalism、SaaS 科技感 |
| 電商 / 消費者應用 | Material Design、有機自然 |
| 金融 / 專業服務 | Minimalism、奢華編輯 |
| 創意 / 設計工具 | Glassmorphism、Brutalism |
| 餐飲 / 生活服務 | 日式禪風、有機自然 |
| 深色環境 / 夜間使用 | 深色模式、SaaS 科技感 |

---

## 1. Minimalism（極簡主義）

**特徵**：大量留白、精簡元素、清晰層級
- **色調**：單色系或雙色系，大面積白/淺灰底
- **排版**：Sans-serif，字級對比強烈，標題粗體突出
- **圓角**：4-8px，低調不搶眼
- **陰影**：極淡或無，靠留白和線條區分層級
- **適用**：B2B、管理後台、專業工具

## 2. Material Design

**特徵**：清晰的層級、有意義的動畫、色彩系統嚴謹
- **色調**：主色+輔色明確，Surface 層級分明
- **排版**：Roboto 或 system-ui，8px grid 嚴格對齊
- **圓角**：8-12px
- **陰影**：Elevation 系統（0/1/2/4/8/16dp）
- **適用**：Android 原生、跨平台、Google 風格產品

## 3. Glassmorphism（毛玻璃）

**特徵**：半透明背景 + 模糊效果 + 微妙邊框
- **色調**：淺色半透明 (rgba白, 0.4-0.7) + 背景模糊 (blur 10-20px)
- **排版**：清晰度要求高，字級不能太小（≥14px）
- **圓角**：12-20px，柔和圓潤
- **邊框**：1px 白色半透明邊框 (rgba白, 0.2)
- **適用**：創意工具、天氣/音樂類、需要「現代感」的產品
- **注意**：對比度容易不足，需仔細驗證 WCAG

## 4. Neumorphism（新擬物）

**特徵**：柔和凸起/凹陷效果，像是從背景「長出來」
- **色調**：單一底色，靠明暗陰影營造立體感
- **排版**：需要足夠粗的字重才能在柔和背景上清晰
- **圓角**：12-16px
- **陰影**：雙向（左上亮 + 右下暗）
- **適用**：概念展示、小型 UI 元件
- **注意**：大面積使用會造成視覺疲勞，對比度問題嚴重

## 5. 日式禪風

**特徵**：極度克制、自然素材質感、不對稱平衡
- **色調**：大地色系（米白、淺灰、木色）+ 一個點綴色（朱紅/深藍）
- **排版**：襯線體（Serif）或明朝體，字距略寬
- **圓角**：0-4px 或完全直角
- **陰影**：極淡或無
- **適用**：餐飲、旅宿、生活風格、高端品牌

## 6. SaaS 科技感

**特徵**：深色系底 + 亮色 accent，數據視覺化導向
- **色調**：深灰/深藍底 + 漸層 accent（藍→紫、綠→青）
- **排版**：等寬字型混搭 sans-serif，數字用 tabular figures
- **圓角**：8-12px
- **陰影**：微光效果（glow），非傳統 drop-shadow
- **適用**：開發者工具、數據平台、監控儀表板

## 7. 奢華編輯（Editorial Luxury）

**特徵**：大圖主導、精緻排版、雜誌感
- **色調**：黑白為主 + 一個金屬色（金/銀）
- **排版**：Serif 標題 + Sans-serif 內文，字級跨度大（48px 標題 / 14px 內文）
- **圓角**：0px（方角為主）
- **陰影**：無，靠留白和對比區分
- **適用**：時尚、奢侈品、高端服務、媒體

## 8. Brutalism

**特徵**：粗獷、大膽、打破常規
- **色調**：高對比（黑+螢光黃、白+紅）
- **排版**：超大字級、不規則排列、混合字型
- **圓角**：0px（銳利方角）
- **邊框**：粗線（2-4px）
- **適用**：藝術、創意、獨立品牌、年輕族群
- **注意**：可讀性較低，不適合工具型產品

## 9. 有機自然

**特徵**：柔和曲線、自然色彩、溫暖親切
- **色調**：大地色（綠、棕、米白）+ 柔和漸層
- **排版**：圓潤的 sans-serif，字距適中
- **圓角**：16-24px（大圓角）
- **陰影**：柔和 diffuse shadow
- **適用**：健康、環保、親子、社群

## 10. 深色模式

**特徵**：深色背景 + 淺色文字/元素
- **色調**：背景 #121212 ~ #1E1E1E（不要純黑 #000000），Surface 用 #1E1E1E ~ #2C2C2C
- **排版**：字重需比亮色模式略輕（Medium → Regular），因為亮字在暗底上會顯粗
- **圓角**：與亮色模式一致
- **陰影**：不使用 drop-shadow（暗底看不到），改用微妙的 border 或 elevation
- **適用**：任何產品的夜間模式、深色環境使用
- **注意**：文字對比用 #E0E0E0 ~ #FFFFFF，避免純白造成眩光
